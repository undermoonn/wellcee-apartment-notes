# Wellcee 房源笔记

一个零构建的 Chrome Manifest V3 扩展，用来给 Wellcee 房源添加仅自己可见的本地笔记。

## 功能

- 在 `/rent-apartment/:id` 详情页的“详情”区块上方显示笔记输入框。
- 输入后自动保存，不需要额外点击按钮。
- 在 `/rent-apartment/:city/list` 列表页，对有笔记的房源卡片在右下角展示单行纯文本摘要，超出宽度时显示省略号。
- 列表页与详情页都可以收藏房源；点击 Chrome 工具栏中的扩展图标可以分别查看收藏列表和有笔记的房源列表。
- 收藏房源后，可以在详情页设置 1 到 5 星评分；取消收藏时会同时删除该房源的评分。
- 扩展弹窗和侧边栏支持按默认顺序或评分从高到低排列收藏、笔记列表；未收藏的笔记房源不能评分。
- 扩展弹窗和侧边栏可以选择在当前标签页或新标签页打开房源，选择会保存在当前 Chrome。
- 可以从扩展弹窗将笔记、房源标题、收藏和评分导出为 JSON，也可以通过同格式文件合并导入。
- 可以从扩展弹窗打开 Chrome 原生侧边栏；侧边栏会在浏览过程中保持开启，并复用完整列表、收藏和数据导入导出能力。
- 当前激活标签页对应的房源会在收藏和笔记列表中突出显示。
- 每次进入已收藏房源的详情页，收藏列表中的标题会自动更新为详情页最新标题。
- 每次进入有笔记房源的详情页，笔记列表中的标题也会自动更新为详情页最新标题。
- 笔记通过 `chrome.storage.local` 保存在当前 Chrome，扩展不会把内容发送到 Wellcee 或其他服务。
- 支持 Wellcee 的懒加载、列表更新和前端路由切换。

## 安装

1. 在 Chrome 地址栏打开 `chrome://extensions`。
2. 打开右上角的“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目目录：`chrome-wellcee-extension`。
5. 刷新已打开的 Wellcee 页面。

## 开发验证

无需安装依赖：

```bash
npm test
```

## 数据说明

笔记按房源 ID 存储在扩展的本地存储中。清空扩展数据或卸载扩展会删除笔记；普通网页脚本无法直接读取这些内容。

## 图标

扩展图标由 OpenAI 内置图像生成工具生成，源文件保存在 `assets/icons/icon-source.png`，并生成了 Chrome 所需的 16、32、48、128 像素版本。

页面内的笔记与收藏图标使用 Google Material Symbols，遵循其开源许可证。
